еұҖеҹҹзҪ‘жҗӯе»әзұ»speedtestжөӢйҖҹзҪ‘з«ҷ librespeed

LibreSpeed жҳҜдёҖдёӘз”ЁдәҺжөӢйҮҸзҪ‘з»ңйҖҹеәҰзҡ„е·Ҙе…·пјҢе®ғзҡ„зү№зӮ№жҳҜз®ҖжҙҒгҖҒжҳ“з”ЁпјҢзү№еҲ«жҳҜеңЁеұҖеҹҹзҪ‘зҺҜеўғдёӯпјҢз”ЁжқҘжөӢиҜ•зҪ‘з»ңи®ҫеӨҮеёҰе®ҪжҳҜеҗҰиҫҫж ҮйқһеёёеҘҪз”ЁгҖӮдҪ еҸҜд»ҘжҠҠе®ғзңӢдҪңspeedtestзҡ„ејҖжәҗз®ҖеҢ–зүҲжң¬гҖӮ

дёҖгҖҒdockerйғЁзҪІ

1.1 й»ҳи®ӨйғЁзҪІ

жӯӨж–№ејҸдёәе®ҳж–№ж–ҮжЎЈдёӯдҪҝз”Ёзҡ„еҚ•жңәжЁЎејҸstandaloneпјҢжңүдёҖдёӘиҮҙе‘ҪзјәзӮ№пјҡжөӢйҖҹз»“жқҹеҗҺзҪ‘йЎөдёӢж–№зҡ„е®ўжҲ·з«ҜipжҳҫзӨәзҡ„жҳҜе®№еҷЁзҡ„иҷҡжӢҹipең°еқҖиҖҢдёҚжҳҜзңҹе®һзҡ„зү©зҗҶжңәip

docker run -e MODE=standalone -p 8080:80 -it ghcr.io/librespeed/speedtest

1.2 жЎҘжҺҘжЁЎејҸйғЁзҪІ

дёәдәҶи§ЈеҶідёҠиҝ°й—®йўҳпјҢеҝ…йЎ»еҗҜз”ЁжЎҘжҺҘжЁЎејҸвҖ“net=hostпјҢеҗҢж—¶дёәйҒҝе…Қ80з«ҜеҸЈе’Ңе®ҝдё»жңәеҶІзӘҒпјҢдҝ®ж”№й»ҳи®ӨзҪ‘йЎөз«ҜеҸЈ80гҖӮ

еҰӮжһңжғідҝ®ж”№жөӢйҖҹзҪ‘йЎөж ҮйўҳпјҢеҸҜд»ҘеҠ дёҠtitleзҡ„еҸҳйҮҸгҖӮ

sudo docker run -e MODE=standalone -e WEBPORT=8999 -e restart=always --net=host ghcr.io/librespeed/speedtest

  • MODE=standalone зӢ¬з«ӢйғЁзҪІжЁЎејҸпјҢ

  • WEBPORT=8999 дҝ®ж”№й»ҳи®ӨзҪ‘йЎөи®ҝй—®з«ҜеҸЈдёә8999

  • restart=always и®ҫзҪ®е®№еҷЁйҡҸе®ҝдё»жңәеҗҜеҠЁ

  • вҖ“net=host дҪҝз”ЁжЎҘжҺҘжЁЎејҸпјҢиҝҷж ·жҳҜдёәдәҶжөӢйҖҹеҗҺжҳҫзӨәе®ўжҲ·з«Ҝipдёәзңҹе®һең°еқҖ

    иҜҰз»ҶеҸӮж•°еҰӮдёӢ

дәҢгҖҒй«ҳзә§жЁЎејҸ

й«ҳзә§жЁЎејҸдёәжңәзҝ»пјҢжң¬дәәжІЎжңүжөӢиҜ•гҖӮиҜ·е°ҪйҮҸзңӢе®ҳж–№ж–ҮжЎЈгҖӮ

еңЁlibrespeedзҡ„е®ҳж–№ж–ҮжЎЈдёӯпјҢиҝҳжҸҗдҫӣдәҶеӨҡзӮ№жөӢиҜ•жЁЎејҸгҖӮиҝҷж—¶йңҖиҰҒ1дёӘд»ҘдёҠзҡ„еҗҺз«ҜжЁЎејҸжңҚеҠЎеҷЁbackendе’Ң1дёӘеүҚз«ҜжңҚеҠЎеҷЁfrontendгҖӮ

2.1 еҗҺз«ҜжЁЎејҸbackend

еңЁеҗҺз«ҜжЁЎејҸдёӢпјҢLibreSpeed д»…жҸҗдҫӣдёҖдёӘжІЎжңү UI зҡ„жөӢиҜ•зӮ№гҖӮдёәжӯӨпјҢиҜ·е°Ҷ MODE зҺҜеўғеҸҳйҮҸи®ҫзҪ®дёә backend гҖӮ

йғЁзҪІеҗҺеҸҜд»ҘеңЁз«ҜеҸЈ 80 дёҠи®ҝй—®д»ҘдёӢеҗҺз«Ҝж–Үд»¶пјҡ garbage.php гҖҒ empty.php гҖҒ getIP.php

2.2 еүҚз«ҜжЁЎејҸfrontend

еңЁеүҚз«ҜжЁЎејҸдёӢпјҢLibreSpeed дёәе®ўжҲ·жҸҗдҫӣ Web UI е’ҢжңҚеҠЎеҷЁеҲ—иЎЁгҖӮдёәжӯӨпјҢиҜ·жү§иЎҢд»ҘдёӢж“ҚдҪңпјҡ

  • и®ҫзҪ® MODE зҺҜеўғеҸҳйҮҸ дёәfrontend

  • еҲӣе»әдёҖдёӘеҢ…еҗ«жөӢиҜ•зӮ№зҡ„servers.jsonж–Үд»¶гҖӮиҜӯжі•еҰӮдёӢпјҡ

  • жіЁж„ҸпјҡеҰӮжһңжңҚеҠЎеҷЁд»…ж”ҜжҢҒHTTPжҲ–HTTPSпјҢиҜ·еңЁжңҚеҠЎеҷЁеӯ—ж®өдёӯжҢҮе®ҡеҚҸи®®гҖӮеҰӮжһңе®ғеҗҢж—¶ж”ҜжҢҒдёӨиҖ…пјҢеҸӘйңҖдҪҝз”Ё // .

  • е®№еҷЁдёӯзҡ„/servers.jsonиҰҒиЈ…иҪҪеҲ°е®ҝдё»жңәпјҲжӯӨж–Үд»¶жң«е°ҫзҡ„зӨәдҫӢпјү

еҸҜд»ҘеңЁз«ҜеҸЈ 80 дёҠи®ҝй—®иҜҘжөӢиҜ•гҖӮ

дёӢйқўжҳҜжӯӨжЁЎејҸдёӢеҸҜз”Ёзҡ„е…¶д»–зҺҜеўғеҸҳйҮҸзҡ„еҲ—иЎЁпјҡ

  • TITLE пјҡйҖҹеәҰжөӢиҜ•зҡ„ж ҮйўҳгҖӮй»ҳи®ӨеҖјпјҡ LibreSpeed
  • TELEMETRY пјҡжҳҜеҗҰеҗҜз”ЁйҒҘжөӢгҖӮй»ҳи®ӨеҖјпјҡ false
  • ENABLE_ID_OBFUSCATION пјҡеңЁеҗҜз”ЁйҒҘжөӢзҡ„жғ…еҶөдёӢи®ҫзҪ®дёә true ж—¶пјҢе°ҶеҜ№жөӢиҜ• ID иҝӣиЎҢжЁЎзіҠеӨ„зҗҶпјҢд»ҘйҒҝе…Қе…¬ејҖж•°жҚ®еә“еҶ…йғЁйЎәеәҸ IDгҖӮй»ҳи®ӨеҖјпјҡ false
  • REDACT_IP_ADDRESSES пјҡеңЁеҗҜз”ЁйҒҘжөӢзҡ„жғ…еҶөдёӢи®ҫзҪ®дёә true ж—¶пјҢе°Ҷд»Һж”¶йӣҶзҡ„йҒҘжөӢж•°жҚ®дёӯзј–иҫ‘ IP ең°еқҖе’Ңдё»жңәеҗҚпјҢд»ҘиҺ·еҫ—жӣҙеҘҪзҡ„йҡҗз§ҒгҖӮй»ҳи®ӨеҖјпјҡ false
  • PASSWORD пјҡи®ҝй—®з»ҹи®ЎдҝЎжҒҜйЎөйқўзҡ„еҜҶз ҒгҖӮеҰӮжһңжңӘи®ҫзҪ®пјҢз»ҹи®ЎдҝЎжҒҜйЎөйқўе°ҶдёҚе…Ғи®ёи®ҝй—®гҖӮ
  • EMAIL пјҡGDPR иҜ·жұӮзҡ„з”өеӯҗйӮ®д»¶ең°еқҖгҖӮеҗҜз”ЁйҒҘжөӢж—¶еҝ…йЎ»жҢҮе®ҡгҖӮ
  • DISABLE_IPINFO пјҡеҰӮжһңи®ҫзҪ®дёә trueпјҢеҲҷдёҚдјҡд»Һ ipinfo.io иҺ·еҸ– ISP дҝЎжҒҜе’Ңи·қзҰ»гҖӮй»ҳи®ӨеҖјпјҡеҖјпјҡ false
  • DISTANCE пјҡи®ҫзҪ®дёә false ж—¶ DISABLE_IPINFO пјҢжҢҮе®ҡеҰӮдҪ•жөӢйҮҸдёҺжңҚеҠЎеҷЁзҡ„и·қзҰ»гҖӮеҸҜд»ҘжҳҜ km е…¬йҮҢгҖҒ mi иӢұйҮҢжҲ–з©әеӯ—з¬ҰдёІпјҢд»ҘзҰҒз”Ёи·қзҰ»жөӢйҮҸгҖӮй»ҳи®ӨеҖјпјҡ km
  • WEBPORT пјҡе…Ғи®ёдёәеҢ…еҗ«зҡ„ Web жңҚеҠЎеҷЁйҖүжӢ©иҮӘе®ҡд№үз«ҜеҸЈгҖӮй»ҳи®ӨеҖјпјҡ 80
